# AudioVault

A modern web-based browser for your audiobook summaries.

## Features
- Grid view of book covers.
- Search by title or author.
- Integrated audio player (supports .opus).
- Markdown transcript viewer.
- Dark theme inspired by Audible.

## Usage
1. Open the [AudioVault](https://sohanuzzaman3301.github.io/audiovault/) website.
2. Click **Select Library** and choose the directory containing your audiobook files.
3. Your books will be displayed automatically.

## Local Privacy
This browser is a static application. All processing happens in your browser, and no data is uploaded to any server.
